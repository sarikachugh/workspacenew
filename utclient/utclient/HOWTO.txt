Building and running the Project using Maven
---------------------------------------

1) Right CLick on the project in eclipse .Go to the Run As menu item and do 

  Maven clean 
  
2) And then repeat the step 1 and do a
  
  Maven install
 
To Run the Client application:

The javafirstwsut should be up and running on the server
 
WSDL URL:
 
http://localhost:8080/javafirstwsut/services/paymentProcessor?wsdl

Running the client:

Run the PaymentWSClient Java class which has the main method.
